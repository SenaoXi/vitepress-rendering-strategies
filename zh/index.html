vitepress-rendering-strategies

赋予 vitepress 多 UI 框架支持

文档介绍
GitHub 仓库
🧩

跨框架渲染

在 VitePress 中原生渲染 React 组件，支持框架隔离、容器级的水合作用（hydration）以及单次属性传递（Vue → React）。

🎛️

渲染策略

提供四种模式：ssr:only (默认)、client:only、client:load 和 client:visible。语义与 Astro 指令对齐。

🔄

spa:sync-render

在 SPA 导航期间同步注入预渲染的 HTML，以消除页面闪烁。适用于关键组件，以平衡用户体验和打包体积。

🔥

开发体验与 HMR

开发与生产环境渲染行为一致，提供完整的热模块重载（HMR）功能，带来流畅的跨框架开发体验。

🌐

SSG/MPA 与 CDN 友好

SSG 优先且兼容 MPA。将预渲染的产物部署到 CDN，以获得更好的 FCP/LCP 和 SEO 表现，同时减轻服务器负载。

🧪

类型安全与测试

严格的 TypeScript 类型定义，并配有单元测试和集成测试，以确保可维护性和稳定性。
